Setup:-
install python
install django
install pycharm

Create first django project

1.create your project directory
2.open cmd prompt in that directory 
3.run django-admin startproject <project-name>

Run django server

1.goto project directory 
2.open project folder/package
3.run python manage.py runserver
4.copy server address in brawoser

create first django app in django project

1.open django project directory
2.open cmd in that dirctory
3.use python manage.py startapp <app-name>
4.check directories
